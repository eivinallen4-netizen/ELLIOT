TOP MOUNTAINEER — BRAND & LOGO KIT
==================================
Palette : Basalt #1A1C20  ·  Snow #F5F5F3  ·  Summit Orange #FF6B2C (accent)
Typeface: Oswald  (wordmark = Bold; tagline = Light, tracked)

The logo is MONOCHROME — your original mountain, one solid color, no accent.
All type is outlined to vector paths, so it renders identically everywhere
with NO font installation required.

LOGOS  (/logos — vector SVG, scales to any size with zero quality loss)
  logo-primary-on-light.svg   Main logo for white / light backgrounds (all Basalt)
  logo-primary-on-dark.svg    Main logo for dark backgrounds (all Snow)
  logo-stacked-on-light.svg   Vertical lockup (light bg) for square-ish spaces
  logo-stacked-on-dark.svg    Vertical lockup (dark bg)
  icon-on-light.svg           Mountain mark only (light bg)
  icon-on-dark.svg            Mountain mark only (dark bg)
  icon-badge.svg              Rounded app-icon / avatar tile
  *-2400.png                  Transparent hi-res PNGs (if a tool won't accept SVG)
  Need pure black/white for a vendor? The on-light/on-dark files are single-color
  already — just swap the fill to #000000 or #FFFFFF in the SVG.

SOCIAL  (/social)
  avatar-1000.png             Profile picture — LinkedIn, TikTok, X, IG
  favicon-512 / 256 / 64.png  Website tab icon (make an .ico from the 256 if needed)

NEWSLETTER  (/newsletter)
  newsletter-masthead-dark.png    1200x300 — drop into the Beehiiv header
  newsletter-masthead-light.png   1200x300 — light-background version

USAGE RULES
  • Clear space: keep at least the height of the mountain clear on all sides.
  • Minimum size: primary logo ~120px wide; icon ~24px.
  • Don't recolor the type, stretch the logo, or add shadows/effects.
  • Summit Orange is your ACCENT color — use it for buttons, links, and highlights
    in your site and newsletter. Keep it OFF the logo itself.
